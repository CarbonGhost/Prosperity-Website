## Mechanics

[Tooled TNT](https://tisunion.github.io/Carpet-TIS-Addition/docs/rules#tooledtnt) causes the effects of tools held by the player to be applied to TNT ignited by said player. This includes gathering specific blocks based on tool, such as grass with shears, as well as applying tool enchantments, such as silk touch or fortune.

[Protection Stacking](https://github.com/manyrandomthings/minitweaks#protectionstacking) allows all protection types to be stacked on a single piece of armor.

[Infinity Mending Stacking](https://github.com/manyrandomthings/minitweaks#infinitymendingstacking) allows both infinity and mending to be placed on a bow.

[Shakable Shulker Boxes](https://github.com/gnembon/fabric-carpet/wiki/Current-Available-Settings#stackableshulkerboxes) enables empty shulker boxes to stack to 64 in inventories or on the ground.

[Blaze Meal](https://github.com/gnembon/carpet-extra#blazemeal) Blaze powder, applied via player or dispenser, fertilizes Nether Wart.

[Flipping Cactus](https://github.com/gnembon/fabric-carpet/wiki/Current-Available-Settings#flippincactus)

[Blaze Meal](https://github.com/gnembon/carpet-extra#blazemeal)

[Placement Rotation Fix](https://github.com/gnembon/fabric-carpet/wiki/Current-Available-Settings#placementrotationfix)

[Ender Pearl Chunk Loading](https://github.com/gnembon/carpet-extra#enderpearlchunkloading)

[Movable Amethyst](https://github.com/gnembon/fabric-carpet/wiki/Current-Available-Settings#movableamethyst) lets players create more efficient amethyst farms by moving budding amethyst blocks using pistons.

## Utilities & Tools

[Bot Players](https://github.com/gnembon/fabric-carpet/wiki/Current-Available-Settings#commandplayer) is a feature which allows you to spawn a "bot" which can be manipulated with commands. This is typically used for AFKing at farms or precisely placing blocks in farms at specific tick intervals, as bots have no risk of being disconnected from the server, nor any connection latency.

[Ping Command](https://github.com/gnembon/carpet-extra#commandping) allows yuo to check your connection latency with the `/ping` command.

[Distance Command](https://github.com/gnembon/fabric-carpet/wiki/Current-Available-Settings#commanddistance)

### Mods Required

[Syncmatica](https://modrinth.com/mod/syncmatica) is an separate extension which allows schematics to be uploaded and synced for multiple members on the server. This is most useful for large scale community projects.
<small>Requires [Litematica](https://www.curseforge.com/minecraft/mc-mods/litematica)</small>

### Creative Only

[WorldEdit](https://modrinth.com/plugin/worldedit) is *the* building utility mod.

[Axiom](https://modrinth.com/mod/axiom) is a building utility mod which focuses on providing a Photoshop-like GUI and in-game overlays.

[Redstone Multimeter](https://modrinth.com/mod/redstone-multimeter) is a mod which allows for players to track a variety of Redstone events with a HUD.

## Social

[Link4D](https://github.com/ProsperityMC/Prosperity-Scripts/blob/main/admin/discord.sc) is an in-house Discord integration used for cross chat between Minecraft and Discord, linking in-game players to Discord accounts, whitelisting, and more.

[Customize](https://github.com/ProsperityMC/Prosperity-Scripts/blob/main/utilities/customize.sc) is a utility we've created to allow players to set and change item lore, names, and textures, utilizing the custom [resource pack](/wiki/resource-pack).

[ItemFlexer](https://modrinth.com/mod/itemflexer) allows players to display their items with a tooltip in chat with `/flex`.

### Mods Required

[Simple Voice Chat](https://modrinth.com/plugin/simple-voice-chat) allows proximity voice chat via Minecraft for all of those who have the mod.

[Audioplayer](https://modrinth.com/mod/audioplayer) is a seperate extension to Simple Voice Chat allowing players to create and play back custom music discs.
<small>Requires [Simple Voice Chat](https://modrinth.com/plugin/simple-voice-chat)</small>

## Misc

[Squaremap](https://modrinth.com/plugin/squaremap) provides a real-time web based map for navigating the server.

[ViaFabric](https://modrinth.com/mod/viafabric) allows players for mismatching Minecraft client versions to join the server.

[TimeOutOut](https://modrinth.com/mod/timeoutout) is a mod which removes the vanilla connection timeout and prevents those players with a poor internet connection from being kicked by the game.
